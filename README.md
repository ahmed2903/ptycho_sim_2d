# ptycho_sim_2d

Simulation of  diffraction patterns from 2D sampels.

It generates synthetic 2D diffraction patterns from a samples and in the far field, enabling researchers to:

- Analyse effects of different lens aberrations
- Produce labelled synthetic datasets for machine learning applications
- Obtain benchmark datasets for testing phase retrieval algorithms

 ## Installation

Clone the repository and install locally:

```bash
git clone https://github.com/ahmed2903/ptycho_sim_2d.git
cd pycbd
pip install .
```
